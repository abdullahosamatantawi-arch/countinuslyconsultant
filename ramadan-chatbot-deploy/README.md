# شات بوت فعاليات رمضان - دائرة الشؤون الإسلامية بالشارقة

## نشر الشات بوت على الإنترنت

### الطريقة 1: Netlify Drop (الأسهل - بدون حساب)

1. اذهب إلى: https://app.netlify.com/drop
2. افتح المجلد: `ramadan-chatbot-deploy`
3. اسحب المجلد وأفلته في صفحة Netlify
4. ستحصل على رابط فوري مثل: `https://your-chatbot.netlify.app`

---

### الطريقة 2: GitHub Pages

1. أنشئ repository جديد على GitHub
2. ارفع محتويات مجلد `ramadan-chatbot-deploy`
3. اذهب إلى Settings > Pages
4. اختر branch `main` واحفظ
5. الرابط سيكون: `https://username.github.io/repository-name`

---

### الطريقة 3: Vercel

1. اذهب إلى: https://vercel.com
2. سجل دخول بحساب GitHub
3. اضغط "New Project"
4. ارفع مجلد `ramadan-chatbot-deploy`
5. احصل على رابط: `https://your-chatbot.vercel.app`

---

## ملاحظات مهمة

- تأكد من وجود ملف `logo.png` في المجلد قبل النشر
- الشات بوت يعمل بدون خادم (Static HTML)
- يمكنك تخصيص اسم الرابط من إعدادات Netlify أو Vercel
