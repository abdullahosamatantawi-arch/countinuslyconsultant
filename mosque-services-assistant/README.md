# المساعد الذكي - إدارة بناء ورعاية المساجد

## نظرة عامة

تطبيق ويب احترافي مصمم خصيصاً لإدارة بناء ورعاية المساجد التابعة لدائرة الشؤون الإسلامية. يوفر التطبيق واجهة عربية كاملة مع تصميم حكومي راقي وثلاث خدمات رئيسية.

## المميزات الرئيسية

### 🕌 الخدمات المتوفرة

1. **زيارة المقابر**
   - حجز موعد لزيارة المقابر
   - نموذج تفصيلي يتضمن: الاسم، رقم الهاتف، تاريخ الزيارة، والموقع

2. **مواقيت الصلاة**
   - عرض مواقيت الصلاة حسب المدينة
   - تحديد تلقائي للصلاة القادمة
   - تحديث تلقائي كل دقيقة

3. **طلب توسعة مسجد**
   - تقديم طلب لتوسعة المساجد
   - نموذج شامل يتضمن: اسم المسجد، الموقع، تفاصيل الطلب، ورقم التواصل

### ✨ مميزات التصميم

- **واجهة عربية كاملة (RTL)** - تصميم يراعي خصوصية اللغة العربية
- **تصميم حكومي احترافي** - ألوان ومظهر يناسب الجهات الحكومية
- **متجاوب بالكامل** - يعمل على جميع الأجهزة (حاسوب، تابلت، هاتف)
- **رسوم متحركة سلسة** - تجربة مستخدم ممتازة
- **نماذج تفاعلية** - تحقق فوري من البيانات
- **إمكانية الوصول** - يدعم التنقل بلوحة المفاتيح

## كيفية الاستخدام

### التشغيل المحلي

1. افتح مجلد المشروع
2. افتح ملف `index.html` في المتصفح
3. أو استخدم خادم محلي:

```bash
# باستخدام Python 3
python -m http.server 8000

# باستخدام Node.js و npx
npx serve
```

### الوظائف الأساسية

#### 1. عرض الخدمات
- عند فتح التطبيق، يتم عرض رسالة ترحيبية
- اختر الخدمة المطلوبة من الأزرار الثلاثة

#### 2. ملء النماذج
- املأ جميع الحقول المطلوبة (المحددة بـ *)
- تحقق تلقائي من صحة البيانات
- رسالة خطأ عند إدخال بيانات غير صحيحة

#### 3. مواقيت الصلاة
- اختر المدينة من القائمة المنسدلة
- عرض تلقائي لمواقيت الصلاة
- تمييز الصلاة القادمة

#### 4. إرسال الطلبات
- بعد ملء النموذج، اضغط على "إرسال الطلب"
- رسالة تأكيد بنجاح الإرسال
- إمكانية العودة للخدمات الأخرى

## التخصيص

### تعديل المدن
في ملف `script.js`، عدّل كائن `prayerTimesData`:

```javascript
const prayerTimesData = {
    'اسم المدينة': {
        fajr: '05:15',
        sunrise: '06:42',
        // ... باقي الأوقات
    }
};
```

### تعديل الألوان
في ملف `styles.css`، عدّل المتغيرات في `:root`:

```css
:root {
    --primary-color: #0a5c44;
    --secondary-color: #c7a86a;
    /* ... باقي الألوان */
}
```

### إضافة خدمة جديدة

1. أضف زر الخدمة في `welcomeScreen`
2. أنشئ شاشة جديدة بمعرف فريد
3. أضف معالج النموذج في `script.js`
