# كيفية رفع المشروع على GitHub (خطوة بخطوة)

**ملاحظة مهمة:** إذا قمت بتثبيت Git والبرنامج مفتوح، **يجب إغلاق VS Code تماماً وإعادة فتحه** لكي يعمل التيرمينال بشكل صحيح.

## 1. فتح التيرمينال (Command Line)
1. من القائمة العلوية في VS Code، اختر **Terminal**.
2. ثم اختر **New Terminal**.
3. (أو اضغط باختصار لوحة المفاتيح: `Ctrl` + `~` حرف الذال).

## 2. التأكد من التثبيت
اكتب الأمر التالي واضغط Enter:
```bash
git --version
```
إذا ظهر لك رقم الإصدار (مثلاً `git version 2.x.x`)، فأنت جاهز للمتابعة. إذا ظهر خطأ، تأكد من إعادة تشغيل البرنامج.

## 3. أوامر الرفع (انسخ والصق)
قم بنسخ هذه الأوامر واحداً تلو الآخر ولصقها في التيرمينال، مع الضغط على Enter بعد كل أمر:

**الخطوة الأولى: تجهيز الملفات**
```bash
git init
git add .
git commit -m "First commit"
```

**الخطوة الثانية: الربط مع GitHub**
(استبدل الرابط أدناه برابط المستودع الذي أنشأته في GitHub)
```bash
git remote add origin https://github.com/ASM-INPUT/mosque-approval-system.git
git branch -M main
git push -u origin main
```
*ملاحظة: سيطلب منك النظام تسجيل الدخول في نافذة منبثقة، قم بإدخال بياناتك.*
